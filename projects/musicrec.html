Music Recommendation Software

  • This program recommends a new music genre to a subscriber.

    It takes in a csv file containing subscriber names and ratings for different genres. It then finds the most similar subscriber and recommends the highest rated genre from that subscriber. It also identifies the most popular genre and calculates the average rating from each genre.

    Unit-testing was conducted to ensure that the program runs for all possible user inputs and dictionaries.

    Utilized: Python, Excel
