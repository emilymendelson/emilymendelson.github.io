Software Projects

  • Move & Restore - Website

    Move & Restore is an web platform allowing for tradition physiotherapy clinics to easily move online. It includes interactive features such as a personalized exercise plan for each patient, a journal feature for patients to record their progress, and a live chat feature for patients to easily communicate with their physiotherapist. Sign up and login features allow users to view a customized exercise plan from their physiotherapist. Confidential patient data is secured using encryption algorithms.

    Utilized: PHP, MySQL Databases, LAMP Stack, CSS, HTML

  • Music Recommendation Software

    This program recommends a new music genre to a subscriber. It takes in a csv file containing subscriber names and ratings for different genres. It then finds the most similar subscriber and recommends the highest genre from that subscriber. It also identifies the most popular genre and calculates the average rating from each genre. Unit-testing was conducted to ensure that the program runs for all possible user inputs and dictionaries.

    Utilized: Python, Excel

  • Vehicle Inventory Management Software

    This program allows users to easily add new vehicles to a data structure. It also allows users to update vehicle information, and perform a keyword-search to view an individual vehicle, or a list of vehicles from the database. A menu was implemented for users to easily navigate through the program.

    Utilized: Python, Excel
